# Responsive Product Grid

A responsive product card layout built using HTML and CSS Grid.  
This project was created to practice modern layout techniques and responsive design for e-commerce style interfaces.

## Technologies Used
- HTML5
- CSS3
- CSS Grid

## Features
- Fully responsive layout
- Modern product card design
- CSS Grid based layout system
- Hover effects and clean UI styling
- Mobile-friendly design

## Purpose of the Project
This project was built as part of my front-end development practice to improve my understanding of responsive layouts and CSS Grid.
